# Portal

Streamline Your Business Operations

## Prerequisites

- Node.js 20+
- Bun 1.1.19+
- Database:PostgreSQL
- Mailing: Resend
- Storage: S3 (R2)

## Development

```bash
cp .env.example .env
bun install
bun run dev
```
